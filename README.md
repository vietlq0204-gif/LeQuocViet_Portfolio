# LeQuocViet_Portfolio

Static portfolio site built with HTML, CSS, and JavaScript.

## Deploy to GitHub Pages

1. Create the GitHub repository or fix the `origin` remote so it points to the correct repo.
2. Push the `main` branch to GitHub.
3. Open `Settings -> Pages` in the GitHub repository.
4. Set `Source` to `GitHub Actions`.
5. Every push to `main` will trigger the workflow in `.github/workflows/deploy-pages.yml`.

The published URL will follow this format:

`https://<github-username>.github.io/<repository-name>/`
